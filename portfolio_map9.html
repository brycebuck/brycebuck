STC Public Comment Cluster Analysis Map Viewer

  • Project Description
  • Every two years, the State Transportation Commission (STC) hosts a public comment period from March to April for the purpose of updating the Twelve-Year Program (TYP). During this time, the public is given the opportunity to complete an online survey or partake in a virtual public meeting to share their transportation-related priorities and concerns with PennDOT. An online mapping component is included where the public can plot points on a webmap to supplement their thoughts. Thousands of georeferenced comments are submitted by the public during each TYP cycle, which are then made available on the Talk PA Transportation website.

    The 2025-2028 TIP: STC Public Comment Cluster Analysis is an ArcGIS Dashboard that houses these georeferenced comments that were made within the SEDA-COG region. I created this tool so one could view the comments spatially, by comment period, and by transportation concern. I wrote an additional report to act as a reference document to the map, which was also included in the 2025-2028 SEDA-COG MPO Transportation Improvement Program (TIP).
