Schuylkilll County Municipal Officials Directory

  • Project Description
  • During my time at Schuylkill County, I was often asked by the public during zoning application reviews for municipal contact information that they could not easily find online. These questions prompted me to create this ArcGIS Experience Build, and accompanying PDF report, that held gathered municipal information in one, easy-to-use repository.

    Within the directory map viewer, one can use the municipality panel to search for municipal contact information. Once a user clicks on a municipality, the map zooms to the area of interest and the contact information is populated on the screen. Open source images of the municipalities were added for greater user context.

    Additionally, this tool was created with updates in mind. A Microsoft Excel sheet was made that could be easily updated, and amended to the existing municipality map layer in ArcGIS Online. It is unknown if this tool will be updated by the County in the future.
